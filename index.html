화면 어디든 눌러주세요!

게임 조작 방법

  • ♥ 왼쪽 상단 메뉴로 게임 상황을 초기화할 수 있어요!

    • 화면 아래 창의 용도는 두가지!

  • 1. 글이 있는 경우

    • ♥ 상황 설명 or 대화창
    • ♥ 다음 화면으로 넘어가려면 박스 터치!
      (넘어가지 않는다면, 글을 터치해주세요!)
    • ♥ 넘어간 지문은 다시 읽을 수 없으니 주의

  • 2. 여러칸이 있는 경우

    • ♥ 획득한 단서를 넣어두는 가방
    • ♥ 자세히 보거나, 단서를 사용하려면 해당 단서의 칸을 터치!
    • ♥ 단서의 다른 면이 존재한다면, 자세히 본 상태에서 단서를 누르기
    • ♥ 단서 사용 - 해당 단서 칸을 터치한 후 사용할 곳을 터치!
      아이템끼리도 사용 가능 ✓

  • ♥ 화살표를 누르면 해당 장소의
    다른 곳을 볼 수 있어요!
    자유롭게 왔다갔다 가능 ✓

  • ♥ 터치하면 정보입력창 등장!
    정답을 창에 입력해주세요

    • 그 외 화면 곳곳 숨겨진 단서는 눌러서 확인해주세요!
